# AI Systems Lab

This repository contains small experiments and prototypes exploring AI systems, LLM workflows, and automation tools.

The goal is to experiment with practical ways AI can augment developer productivity and backend systems.

---

## Areas of Exploration

- LLM routing and fallback strategies
- prompt evaluation and scoring
- AI-assisted developer tooling
- workflow automation
- CLI-based AI tools
- retrieval and context systems

---

## Experiments

Examples of experiments that may appear in this lab:

- multi-model routing strategies
- prompt scoring systems
- small agent workflows
- developer productivity automation
- LLM-assisted code analysis

---

## Philosophy

The purpose of this repository is experimentation.

Ideas here may evolve into full systems such as:

- CodeCrusher
- other developer tooling
- AI automation platforms
